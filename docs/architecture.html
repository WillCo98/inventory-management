Factory Inventory Management System

Vue 3 + FastAPI full-stack architecture with in-memory mock data

Tech Stack

Frontend (port 3000)

  • Vue 3 + Composition API
  • Vue Router 4 (6 routes)
  • Vite build tool
  • Axios HTTP client
  • Custom i18n (EN / JA)
  • Scoped CSS + CSS Grid

Backend (port 8001)

  • Python FastAPI
  • Uvicorn ASGI server
  • Pydantic v2 models
  • CORS middleware
  • 20+ REST endpoints
  • Auto-generated /docs

Data Layer

  • 7 JSON data files
  • In-memory at runtime
  • No database (demo)
  • Non-persistent writes
  • Loaded via mock_data.py
  • generate_data.py for seeding

Key Design Decisions

No database -- All data in-memory from JSON; non-persistent, reloads on restart
Singleton filters -- Global state via useFilters composable, not Vuex/Pinia
No authentication -- Hardcoded user; demo-only with CORS wildcard
Quarter support -- Backend maps Q1-2025 to [2025-01, 2025-02, 2025-03]
Custom SVG charts -- No charting library; donut, bar, and stacked bar via SVG
Modal Teleport -- Teleport to body prevents CSS stacking context issues
Computed currency -- Switches USD/JPY automatically with locale change
Case-insensitive filters -- Backend lowercases categories during comparison
